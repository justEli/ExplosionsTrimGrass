# ExplosionsTrimGrass

Explosions, including creepers, will turn grass blocks into dirt, instead of breaking the blocks.

With this plugin, when a creeper explodes above grass blocks, it will not break the blocks, but instead turn grass blocks into dirt and coarse dirt. It will also break snow, flowers and crops. This is all configurable, however.

- [SpigotMC](https://www.spigotmc.org/resources/explosionstrimgrass.73305/)
- [Hangar (Paper)](https://hangar.papermc.io/Eli/ExplosionsTrimGrass)
- [Modrinth](https://modrinth.com/plugin/explosionstrimgrass)

## Platforms supported
- Spigot ✔️
- Paper ✔️
- Folia ✔️
